## Code Snippets

### Problem 1
```java
import java.util.HashSet;

public class Main {
    public static void main(String[] args) {
        HashSet<String> set = new HashSet<>();
        set.add("apple");
        set.add("banana");
        set.add("orange");

        System.out.println("HashSet elements: " + set);
    }
}
```
